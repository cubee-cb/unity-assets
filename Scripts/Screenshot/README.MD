# Screenshot script
Take a 4096x4096 screenshot of the scene from a camera's perspective. Useful for creating thumbnail images of assets.

How-To:
- Add this script to a camera.
- Enter an identifier for this camera's image filename. (e.g. could be `main_image`, `turnaround`, and `pc_mobile_compare` for a character model)
- Press the button. An image will be captured.
  - You can also set "Capture on Start" to capture an image upon entering Play Mode.
- Captured images will be in `Assets/Screenshots`. Existing images will be overwritten as you take new photos.
