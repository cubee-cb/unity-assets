# Unity Assets
Various assets or tools I've made for use in Unity.

## Scripts
- [Screenshot](Scripts/Screenshot) - Take a transparent screenshot of the scene from a camera's perspective.
- [MeshBaker](Scripts/MeshBaker) - Combine Skinned and normal Meshes into a single regular mesh.

## Shaders
- [retrop8](Shaders/retrop8) - A shader based on the rendering style of [picoCAD](https://johanpeitz.itch.io/picocad) and [PICO-8](https://www.lexaloffle.com/pico-8.php).
